# VCC Sesión 3 — M2 Proyectos + M3 Gobernanza — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add real main content to the VCC — M2 (project catalog with VS Code launcher) and M3 (governance script runner with live WebSocket output), plus real tunnel status in sidebar.

**Architecture:** WebSocket server attached to existing Express HTTP server (same port 8080). Three new backend routes (tunnels, projects, govern). Frontend gets tabs nav, M2 accordion by client, M3 terminal panel with WebSocket client.

**Tech Stack:** Node.js ES modules, Express 4, ws (WebSocket), vanilla JS, no frameworks.

## Global Constraints

- No new npm packages except `ws`
- `WORKSPACE_ROOT = 'C:\\Users\\Carlos Valera\\OneDrive\\Escritorio\\AI-Workspace'`

- Vanilla JS — no framework, no bundler
- JetBrains Mono, CSS custom properties already defined in style.css
